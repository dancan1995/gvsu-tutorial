Descriptive Statistics

Steps in SPSS:

  1. Go to Analyze > Descriptive Statistics > Frequencies.
  2. Select the variables you want to summarize (e.g., age, weight, gender).
  3. Click OK to generate frequencies or descriptive statistics.

Example Output:

Variable Mean Standard Deviation Minimum Maximum
Age 45.6 12.4 18 76
Weight 78.5 15.3 50 110

Interpretation: The average age of the participants is 45.6 years, with a range between 18 and 76. The mean weight is 78.5 kg, with a standard deviation of 15.3 kg, indicating some variability in weight among participants.

Independent Samples T-Test

Steps in SPSS:

  1. Go to Analyze > Compare Means > Independent-Samples T-Test.
  2. Select your test variable (e.g., systolic blood pressure) and grouping variable (e.g., gender).
  3. Define the groups (e.g., 1 = Male, 2 = Female).
  4. Click OK to run the test.

Example Output:

Group Mean Blood Pressure Std. Deviation Sig. (2-tailed)
Male 130 15 0.05
Female 125 14

Interpretation: The independent t-test reveals a significant difference in blood pressure between males (M = 130) and females (M = 125), p = 0.05. This suggests that males have a higher average blood pressure than females.

Paired Samples T-Test

Steps in SPSS:

  1. Go to Analyze > Compare Means > Paired-Samples T-Test.
  2. Select the pair of variables to compare (e.g., pre- and post-intervention blood pressure).
  3. Click OK to run the test.

Example Output:

Time Mean Blood Pressure Std. Deviation Sig. (2-tailed)
Pre-intervention 140 10 0.01
Post-intervention 130 9

Interpretation: The paired t-test shows a significant decrease in blood pressure following the intervention (pre: M = 140, post: M = 130, p = 0.01). This indicates that the intervention effectively reduced blood pressure.

One-Way ANOVA

Steps in SPSS:

  1. Go to Analyze > Compare Means > One-Way ANOVA.
  2. Select your dependent variable (e.g., systolic blood pressure).
  3. Select your factor (grouping) variable (e.g., age group: young, middle-aged, older).
  4. Click OK to run the test.

Example Output:

Source Sum of Squares df Mean Square F Sig.
Between Groups 300 2 150 5.6 0.01
Within Groups 2000 197 10.15
Total 2300 199

Interpretation: The ANOVA results show a statistically significant difference in blood pressure between age groups, F(2, 197) = 5.6, p = 0.01. This indicates that blood pressure varies significantly by age group.

Chi-Square Test

Steps in SPSS:

  1. Go to Analyze > Descriptive Statistics > Crosstabs.
  2. Place the categorical variable in the Row(s) and Column(s) fields.
  3. Click on Statistics and check Chi-square.
  4. Click OK to run the test.

Example Output:

Crosstabulation

Gender Smoker Non-Smoker Total
Male 50 30 80
Female 20 100 120
Total 70 130 200

Chi-Square Test Results

Test Value Degrees of Freedom p-value
Chi-Square 12.24 1 0.0005

Interpretation: The Chi-Square test indicates a significant association between gender and smoking status (χ²(1, N = 200) = 12.24, p < 0.001). This suggests that smoking habits differ between males and females, with a higher proportion of males being smokers compared to females.

Correlation Analysis

Steps in SPSS:

  1. Go to Analyze > Correlate > Bivariate.
  2. Select the variables you want to analyze (e.g., hours studied and test scores).
  3. Check Pearson and click OK.

Example Output:

Correlation Matrix

Variable Hours Studied Test Score
Hours Studied 1.00 0.85
Test Score 0.85 1.00

Interpretation: There is a strong positive correlation between hours studied and test scores (r = 0.85, p < 0.001). This suggests that as study hours increase, test scores tend to increase as well.

Simple Linear Regression

Steps in SPSS:

  1. Go to Analyze > Regression > Linear.
  2. Select the dependent variable (e.g., test score) and the independent variable (e.g., hours studied).
  3. Click OK to run the regression analysis.

Example Output:

Regression Coefficients Table

Variable B Std. Error t p-value
Intercept 50.00 3.00 16.67 0.000
Hours Studied 5.00 0.50 10.00 0.000

Interpretation: The regression analysis shows that for each additional hour studied, the test score increases by 5 points (B = 5.00, p < 0.001). The model is statistically significant, indicating a good fit for the data.

Multiple Linear Regression

Steps in SPSS:

  1. Go to Analyze > Regression > Linear.
  2. Select the dependent variable (e.g., test score) and multiple independent variables (e.g., hours studied, attendance).
  3. Click OK to run the regression analysis.

Example Output:

Regression Coefficients Table

Variable B Std. Error t p-value
Intercept 40.00 4.00 10.00 0.000
Hours Studied 3.50 0.70 5.00 0.000
Attendance 2.00 0.50 4.00 0.001

Interpretation: The multiple regression analysis indicates that both hours studied (B = 3.50, p < 0.001) and attendance (B = 2.00, p = 0.001) significantly predict test scores. This suggests that higher study hours and better attendance are associated with higher test scores.

Binary Logistic Regression

Steps in SPSS:

  1. Go to Analyze > Regression > Binary Logistic.
  2. Select the dependent variable (e.g., outcome: disease presence) and independent variables (e.g., age, gender).
  3. Click OK to run the logistic regression.

Example Output:

Variables in the Equation

Variable B Wald df p-value Exp(B)
Intercept -1.50 5.00 1 0.025
Age 0.05 4.00 1 0.045 1.05
Gender (Male) 0.80 6.50 1 0.011 2.22

Interpretation: The binary logistic regression analysis shows that age (B = 0.05, p = 0.045) and being male (B = 0.80, p = 0.011) significantly predict the likelihood of having the disease. The odds of having the disease increase by 5% for each additional year of age, and males have approximately 2.22 times the odds of having the disease compared to females.

Kaplan-Meier Survival Analysis

Steps in SPSS:

  1. Go to Analyze > Survival > Kaplan-Meier.
  2. Select the time variable (e.g., survival time) and status variable (e.g., censored or event occurrence).
  3. Define the event status by coding (e.g., 1 for event occurred, 0 for censored).
  4. Under Options, choose to display the survival plot and log-rank test if needed.
  5. Click OK to run the analysis and generate the survival curve.

Example Output:

Survival Table

Time (Months) Number at Risk Number of Events Survival Probability
1 100 2 0.98
3 98 8 0.90
6 90 15 0.75
12 75 20 0.55

Interpretation: The Kaplan-Meier analysis shows that the survival probability decreases over time. After 12 months, the estimated survival probability is 55%, meaning 55% of the subjects are expected to survive past 12 months. The number of events (deaths or other endpoints) increases as time progresses, leading to a decline in survival probability.

Based on the Kaplan-Meier survival curve, the survival rate drops sharply between 6 and 12 months, indicating that this period might be critical in the study population. Further analysis, such as the log-rank test, can be conducted to compare survival curves across different groups (e.g., treatment vs. control).
